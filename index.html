External Resources

w3schools

The de-facto site for information on web technologies (in my opinion, at least). In addition to having expansive documentation and examples, it also has number of informative tutorials and exercises for programming. There are also a number of useful statistics for web developers, like what browsers are most popular.

  • HTML element reference
  • HTML attribute reference
  • HTML examples
  • CSS reference
  • CSS examples
  • Browser usage statistics
  • JavaScript tutorial

Mozilla Developer Network

Maintained by Mozilla, the company that develops the Firefox browser. There are a number of useful resources available on this site, including:

  • Documentation for HTML
  • A list of available HTML elements, with in depth descriptions, browser support information, and demos
  • Documentation for CSS, including a list of available styling rules, browser support information, and demos
