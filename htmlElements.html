HTML Elements

All HTML elements have the following optional attributes:

  • class Used to give a list of groups that the element belongs to.
  • id Used to give the element a unique identification.
  • title Used to give the 'title text' of an element. This is usually interpreted as mouse-over text or a tool tip.

A list of commonly used HTML elements is given below

  • <a>

    • Stands for Anchor
    • Use Making links
    • Additional attributes
      • href="string" The location or URL that the link goes to. This can be a website, image, email, phone number, HTML element, file, or more. Basically, if it has a internet address of some sort, you can link to it.
      • download(="string") Specifies that the file should be downloaded instead of opened in the browser. If a string value is given, it will be used as the default file name of the download
      • target="string" Used to open the link in a pop-up window. This is usually considered rude, so avoid it unless it is absolutely necessary.
